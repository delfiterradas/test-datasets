# Candidatus Portiera aleyrodidarum - gapseq test data

## Overview

This directory contains test data for **gapseq** metabolic model reconstruction modules and **memote** quality control validation.

**Organism**: *Candidatus Portiera aleyrodidarum*  

## Files

### Input files (gapseq intermediate outputs)

These files are generated by `gapseq find` and `gapseq findtransport` commands:

- **`proteome-all-Reactions.tbl`** (1.5M) - Predicted metabolic reactions from proteome annotation
- **`proteome-all-Pathways.tbl`** (146K) - Predicted metabolic pathways 
- **`proteome-Transporter.tbl`** (136K) - Predicted transport reactions

### Model files

- **`test.xml`** (1.5M) - Gap-filled genome-scale metabolic model in SBML format
  - Generated by `gapseq doall` (complete workflow: find + draft + fill)
  - SBML Level 3 Version 2 with FBC (Flux Balance Constraints) package
  - Ready for flux balance analysis (FBA) and quality control with memote

## How files were generated

The complete workflow used gapseq v2.0.1:

```bash
# Step 1: Find reactions and pathways from proteome
gapseq find -p all -t Bacteria proteome.fasta
gapseq findtransport -t Bacteria proteome.fasta

# Step 2: Draft model and gap-filling (generates both .RDS and .xml)
gapseq draft -r proteome-all-Reactions.tbl \
             -t proteome-Transporter.tbl \
             -p proteome-all-Pathways.tbl \
             -n test

gapseq fill -m test-draft.RDS -n test

# Alternative: Single command (generates test.xml automatically)
gapseq doall -t Bacteria proteome.fasta
```
