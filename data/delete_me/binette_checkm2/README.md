# Database for Binette/CheckM2

The file included here is a very small subset of the CheckM2 database, intended for use with Binette.
It consists of a small DIAMOND database for gene annotation.

Unfortunately it is not currently compatible with the actual CheckM2 script, as this includes an MD5
check on the database file. I have named this directory prospectively though in the hope that this
check can be relaxed in future.
