# Test data for METATOR

https://github.com/koszullab/metaTOR

All data copied from the metator tests directory: https://github.com/koszullab/metaTOR/tree/master/tests_data

- assembly.fa - assembly FASTA for binning
- {for,rev}_paired.fq.gz - PE Hi-C reads
- alignment_0_{for,rev}.bam - Hi-C alignments
- alignment.pairs.gz - Hi-C alignments in pairs format
- depths.txt - contig depths TSV
