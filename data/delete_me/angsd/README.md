# ANGSD Test Data

## Data Description

- dosaf
    - 'angsd_FIN.indf': Per-individual inbreeding coefficients for FIN individuals HG00349, HG00350, HG00351, HG00358. Values are drawn from Gazal et al. (2015) (Scientific Reports, doi:10.1038/srep17453), which estimated genome-wide inbreeding coefficients for all individuals in the final 1000 Genomes Project release. 
    - 'angsd_syk.errors': Randomly generated base substitution error matrix for ANGSD's SYK GL model (-GL 4). Not derived from real data; for testing purposes only.
